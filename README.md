## Description

This Project is a React Test Project for Addis Software. It is a simple React App using React, Redux, Redux Saga, Emotion, NodeJS, Express, MongoDB, Mongoose.

## Functionalities

- Add Music
- Update Music Info
- Delete Music
- Get Music
- Search music by music title and music author
